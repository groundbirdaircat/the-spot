The Spot
The main focus of this project was to test the Canvas API with procedural map generation, selective rendering, multiplayer websocket connections, and dynamic items.

This is not a fully polished game, and there is no progress, goal, or end point.

Then what is it? The Spot is an explorable chat environment with a new map generated daily. Each map contains thousands of searchable houses, in which you can find one type of (useless) item: water.

Pick a color and a name, and then press start to check it out.
